# 🔢 수치 키워드 검색량 조회기 v1.0

> **만든 날짜:** 2026-03-27
> **경로:** `프로그램 개발\수치키워드_검색량\`
> **실행:** `실행.bat` 더블클릭

---

## 🎯 이 프로그램이 하는 일

건강 수치 키워드(혈압, 당화혈색소, 콜레스테롤 등)에 **숫자를 자동으로 붙여서** 네이버 검색량을 일괄 조회합니다.

> 예시: "혈압" → "혈압 40", "혈압 41", ... "혈압 300" 전부 검색량 확인

---

## ✅ 잘된 점

| 항목 | 설명 |
|------|------|
| **자동 조합 생성** | 키워드 + 숫자 범위만 지정하면 수백~수천 개 조합 자동 생성 |
| **소수점 지원** | 당화혈색소 3.0, 3.1, ... 15.0 같은 소수점 단위 조회 가능 |
| **두 숫자 조합** | "혈압 140 90" 같은 수축기/이완기 조합도 지원 |
| **중복 조회 방지** | 이미 조회한 키워드는 건너뛰고 새로 추가된 것만 조회 |
| **시트 기반 설정** | 구글 시트에서 키워드/범위 수정 → 비개발자도 관리 가능 |

---

## 💡 사용 방법

### 1단계: 시트에서 설정 확인/수정

시트 `수치키워드_설정` 탭:

| 키워드 | 시작 | 끝 | 간격 | 시작2 | 끝2 | 간격2 |
|--------|------|-----|------|-------|-----|-------|
| 혈압 | 40 | 300 | 1 | | | |
| 혈압 | 80 | 200 | 10 | 40 | 130 | 10 |
| 당화혈색소 | 3 | 15 | 0.1 | | | |

- **E~G열이 비어 있으면**: 단일 숫자 (혈압 120)
- **E~G열이 채워져 있으면**: 두 숫자 조합 (혈압 140 90)
- 키워드/범위를 자유롭게 추가/수정 가능

### 2단계: 실행

`실행.bat` 더블클릭 → 자동 조회 → `수치키워드_결과` 탭에 기록

### 3단계: 결과 확인

`수치키워드_결과` 탭:

| 기본 키워드 | 수치 키워드 | 검색량 |
|-------------|-------------|--------|
| 혈압 | 혈압 정상수치 | 12,000 |
| 혈압 | 혈압 120 | 5,400 |
| ... | ... | ... |

> 검색량 높은 순으로 정렬됩니다.

---

## ⚠️ 주의사항

- **API 속도 제한**: 0.5초 간격으로 5개씩 조회. 조합이 많으면 시간이 걸림
  - 예: 1,000개 조합 → 약 100초(1분 40초)
- **두 숫자 조합 간격**: 1 단위로 하면 조합 수가 폭발 → **10 단위 권장**
- **새 키워드 추가 시**: 설정 탭에 행 추가 후 다시 실행하면 신규만 조회
