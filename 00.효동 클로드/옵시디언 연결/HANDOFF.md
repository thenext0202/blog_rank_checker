## 구현 완료 (2026-03-20)

### 핵심 기능
1. **폴더 추가** — 폴더 선택 시 하위 지원 파일 전부 자동 수집
2. **파일 추가** — 개별 파일 직접 선택
3. **체크박스 선택** — 파일별 변환 여부 개별 선택/해제
4. **전체 선택/해제** — 일괄 토글
5. **선택 항목 제거** — 체크된 파일만 목록에서 제거
6. **저장 폴더 지정** — 볼트 내 하위 폴더 자동 생성
7. **프론트매터 옵션** — 원본 경로, 파일 유형, 변환일시, 크기 메타데이터

### 지원 파일 형식 & 변환 방식
| 분류 | 확장자 | 변환 방식 |
|------|--------|-----------|
| 코드 | .py .js .ts .java .go .sql .html .css .yaml 등 20+ | 언어 표시 + 코드 블록 |
| 문서 | .docx | 제목/본문/볼드·이탤릭/표 추출 |
| PDF | .pdf | 페이지별 텍스트 추출 |
| 스프레드시트 | .xlsx .xls | 시트별 마크다운 표 (최대 100행 미리보기) |
| 데이터 | .csv | 마크다운 표 변환 |
| JSON | .json | 구조 요약 + pretty print 코드 블록 |
| 텍스트 | .txt .md .log | 그대로 또는 텍스트 정리 |

### GUI
- tkinter + ttk (clam 테마, 모던 라이트 스타일)
- 폴더별 그룹핑된 파일 목록 (스크롤 가능)
- 파일 아이콘 (유형별 이모지 구분)
- 변환 진행 상태 표시 (스레드 처리)
- 중복 파일명 자동 넘버링

### 외부 의존성
| 패키지 | 용도 | 필수 여부 |
|--------|------|-----------|
| openpyxl | .xlsx 읽기 | .xlsx 사용 시 |
| python-docx | .docx 읽기 | .docx 사용 시 |
| PyPDF2 또는 pypdf | .pdf 읽기 | .pdf 사용 시 |

> 미설치 시 해당 파일 변환에서 안내 메시지 표시 (프로그램 자체는 정상 동작)
