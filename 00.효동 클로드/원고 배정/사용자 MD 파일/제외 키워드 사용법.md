# 🔒 업체별 제외 키워드 기능

> 특정 업체에 배정하면 안 되는 키워드를 미리 등록해 두는 기능
> 등록한 키워드가 원고 **제목**이나 **폴더명**에 포함되면, 해당 업체에는 자동 배정되지 않음

### 💡 예시
> 긍정애드에 `탈모`, `병원`을 등록하면
> → "탈모 예방 습관" 같은 원고는 긍정애드에 배정 안 됨
> → 다른 업체에 자동으로 배정됨

---

<br>

# 📖 사용 방법

### 1️⃣ 키워드 등록
1. 프로그램 상단의 **"제외 키워드 설정"** 버튼 클릭
2. 업체 선택 (콤보박스)
3. 키워드 입력 후 **추가** 클릭
   - 쉼표(`,`)로 여러 개 한번에 입력 가능 → `탈모, 병원, 수술`
   - `Enter` 키로도 추가 가능

### 2️⃣ 키워드 삭제
1. 목록에서 삭제할 키워드 클릭
2. **선택 삭제** 버튼 클릭

### 3️⃣ 키워드 수정
- 직접 수정은 안 됨 → **삭제 후 다시 추가**

### 4️⃣ 배정 결과 확인
- Step 3 (자동 배정) 화면의 **"제외"** 컬럼에 표시됨
- 예: `긍정애드(탈모)` → 긍정애드가 "탈모" 때문에 제외됐다는 뜻
- 제외된 항목은 🔴 **빨간 배경**으로 표시

### 5️⃣ 저장 위치
- `exclude_keywords.json` 파일이 프로그램 옆에 자동 생성됨
- 프로그램을 껐다 켜도 유지됨

# 🔧 구현 방식 (수정/제거 시 참고)

> 이 기능을 빼거나 수정하고 싶을 때 아래 표를 보고 해당 부분만 찾으면 됨

### 📍 추가된 코드 위치 (`main.py`)

| 위치 | 내용 |
|:-----|:-----|
| 상단 `import` | `json` 추가 |
| 상수 영역 | `EXCLUDE_FILE` 경로 상수 |
| 유틸리티 섹션 | `load_exclusions()` — JSON 파일 읽기 |
| | `save_exclusions()` — JSON 파일 쓰기 |
| | `check_exclusion()` — 제목/폴더명에 제외 키워드 포함 여부 판정 |
| `__init__` | `self.exclusions = load_exclusions()` 로드 |
| `_build_ui` (상단 바) | "제외 키워드 설정" 버튼 |
| `_open_exclusion_dialog()` | 설정 다이얼로그 (Toplevel 팝업) |
| `_auto_assign()` | 제외 키워드 매칭 시 해당 업체 `continue` 스킵 |
| | 제외된 업체 목록을 "제외" 컬럼에 표시 |
| `_change_company()` | 수동 변경 시에도 제외 경고 표시 |
| Step 3 트리뷰 | `"excluded"` 컬럼/태그 추가 (빨간 배경 `#FFCDD2`) |

### ⚙️ 제외 판정 로직
```python
check_exclusion(exclusions, company, title, folder_name)
# → 업체의 제외 키워드 목록을 순회
# → (제목 + 폴더명)을 합쳐서 소문자로 비교
# → 포함되면 해당 키워드 문자열 반환, 아니면 None
```

### ⚙️ 자동 배정에서의 동작
```python
for company in 후보업체:
    if check_exclusion(해당업체, 원고제목, 폴더명):
        continue  # ← 이 업체는 아예 건너뜀
    ... 기존 점수 계산 ...
```
> ⚠️ 제외된 업체는 점수 계산 자체를 안 함 (감점이 아니라 **완전 스킵**)

### 📄 JSON 파일 구조
```json
{
  "긍정애드": ["탈모", "병원"],
  "궁서": ["수술"]
}
```

# ✅ 잘된 점
| 항목 | 설명 |
|:-----|:-----|
| **영구 저장** | JSON 파일로 저장 → 프로그램 재시작해도 유지 |
| **쉼표 입력** | 여러 키워드를 한번에 등록 가능 |
| **완전 스킵** | 감점이 아니라 후보에서 완전 제외 → 실수 배정 불가 |
| **시각적 표시** | 빨간 배경 + 어떤 업체가 어떤 키워드로 제외됐는지 즉시 확인 |

# ⚠️ 아쉬운 점 / 개선 가능

| 문제 | 현재 | 개선 방향 |
|:-----|:-----|:----------|
| **부분 매칭만 지원** | "탈모" 등록 시 "탈모예방"도 걸림 | "포함" vs "정확히 일치" 옵션 추가 |
| **인라인 수정 없음** | 오타 시 삭제 후 재추가 | 더블클릭으로 수정 |
| **전체 보기 없음** | 업체 하나씩만 확인 가능 | 모든 업체 한눈에 보는 테이블 뷰 |
| **제외 사유 미기록** | 왜 제외했는지 메모 불가 | 키워드 옆에 사유 입력란 |
