# 📂 원고 폴더 열기 + ⚠ 시트 기입 취소 기능

> **날짜**: 2026-03-24
> **프로그램**: 원고 배정 프로그램 v1.4+

---

<br>

## 🔧 추가된 기능 2가지

| # | 기능 | 위치 | 동작 |
|:-:|:-----|:-----|:-----|
| 1 | **원고 폴더 열기** | Step 2 매칭 목록 | 파일명 **더블클릭** → 탐색기로 폴더 열림 |
| 2 | **시트 기입 취소** | 하단 네비게이션 바 | 시트에 기입한 행을 삭제하여 되돌림 |

<br>

## 📂 1. 원고 폴더 열기

### 사용 방법
1. Step 2에서 매칭+검수 완료 후 목록이 표시됨
2. 확인하고 싶은 원고의 **파일명(폴더명)을 더블클릭**
3. Windows 탐색기로 해당 원고 폴더가 열림

### 주의사항
- "시트만" 항목처럼 폴더가 없는 경우 → **"폴더 없음" 경고 팝업** 표시
- 단순 클릭(선택)은 기존대로 하단 검수 상세 패널에 정보 표시

---

<br>

## ⚠ 2. 시트 기입 취소

### 사용 방법
1. Step 1에서 "시트 기입 + 불러오기" 실행 → 기입 완료 후 **"⚠ 시트 기입 취소" 버튼 활성화**
2. 어느 Step에서든 하단의 버튼 클릭
3. 확인 팝업 → "예" → 자사 발행리스트에서 기입한 행 삭제

### 동작 원리

```
시트 기입 시: 기입한 행 번호를 written_rows에 기록
취소 시:     큰 번호부터 역순 삭제 (행 번호 밀림 방지)
            → publish_data 초기화
            → 버튼 비활성화
```

### 취소 가능 범위

| 상황 | 취소 방법 |
|:-----|:----------|
| Step 1 시트 기입 후 ~ Step 3 | **"⚠ 시트 기입 취소"** 버튼 (하단 바) |
| Step 4 배정 실행 후 | **"⚠ 작업 모두 취소"** 버튼 (Step 4 내부, 파일 복원 + 시트 행 삭제 포함) |

> Step 4의 "작업 모두 취소" 완료 시 written_rows도 함께 초기화됩니다.

## ✅ 잘된 점
- **조기 취소 가능** — 시트 기입 직후부터 되돌릴 수 있어 실수 복구가 빠름
- **하단 고정 위치** — 어느 Step에서든 접근 가능
- **폴더 열기** — 검수 실패 원고를 바로 확인할 수 있어 작업 효율 향상

## ⚠️ 아쉬운 점
- 시트 기입 취소 후 Step 1 Treeview는 자동 갱신되지 않음 (새로고침 또는 재실행 필요)
- 다른 세션에서 기입한 행은 취소 대상에 포함되지 않음 (현재 세션 기입분만)
