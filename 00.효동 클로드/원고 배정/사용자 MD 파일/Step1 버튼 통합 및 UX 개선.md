## 🚀 사용 방법

### Step 1 — "시트 기입 + 불러오기" 버튼

1. **업체별 건수 입력 후 클릭** → 시트에 기입 → 자동으로 발행리스트 불러오기
2. **건수 모두 0으로 두고 클릭** → 시트 기입 없이 기존 데이터 바로 불러오기
3. **해당 날짜에 이미 데이터 있으면** → "기존 데이터를 불러올까요?" 확인 팝업 → "예" 클릭 시 불러오기

> 💡 기존에 2번 눌러야 했던 것이 **1번으로 줄어듦**

### Step 4 — 배정 실행

- "배정 실행" 클릭 → 파일 복사/ZIP/삭제 → **시트 파라미터 업데이트 + 키워드 시트 기입까지 자동 진행**
- 파라미터가 없어도 키워드 시트 기입은 실행됨

## 🔧 구현 방식

### `_write_plan_and_load()` (기존 `_write_plan_to_sheet` 대체)

```
건수 확인
├─ 건수 0 → _load_publish_data() 바로 호출
├─ 건수 > 0 → 확인 팝업
│   ├─ 시트에 기존 데이터 있음 → 확인 팝업 → "예" → _load_publish_data()
│   └─ 시트에 기존 데이터 없음 → 기입 → 자동 _load_publish_data()
```
