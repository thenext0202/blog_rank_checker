# 🏗️ 프로그램 개요

원고를 업체별로 자동 배정하는 tkinter GUI 프로그램.
발행 계획 설정 → 매칭+검수 → 자동 배정 → 실행(복사/ZIP/삭제) 4단계로 동작.

# 🔄 4단계 워크플로우

### Step 1. 배정 건수 설정
- 업체별 건수 스핀박스 → **"시트 기입 + 불러오기" 버튼 1개로 통합**
- 건수 > 0: 시트 기입 → 자동 발행리스트 불러오기
- 건수 0: 기입 스킵, 바로 기존 발행리스트 불러오기
- 해당 날짜에 기존 데이터 있으면 확인 팝업 → "예" 시 불러오기
- H열 기준 마지막 행 다음에 기입 (수식 보호, A+N열만 기입)

### Step 2. 매칭 + 검수
- 🔗 **Step 2 진입 시 자동으로 매칭+검수 실행** (수동 재실행 버튼 유지)
- 원고리스트 AG열 ↔ 미배정 폴더명 매칭 (정확 → 부분 매칭)
- ZIP 자동 압축 해제
- **검수 항목**: 이미지 번호(30 이하만 인식), 링크, 광고이미지
- 병렬 검수 (8개), 수동 통과 처리, 컬럼 정렬
- **📌 기인정**: 수동 통과한 원고를 `manual_pass.json`에 영구 저장 → 다음 실행 시 자동 통과("기인정" 표시)
- 📂 **파일명 더블클릭 → 원고 폴더 탐색기로 열기** (폴더 없으면 경고 팝업)

### Step 3. 자동 배정
- 제품 매칭 우선 → 남은 건수 → 5일 키워드 겹침 패널티
- **🔒 제외 키워드**: 업체별 등록된 키워드가 제목/폴더명에 포함되면 완전 스킵
- 수동 업체 변경 + "(선택 안 함)" 제외
- 재배정 가능

### Step 4. 실행
- 원고모음 복사 + 발행요청 ZIP + 미배정 삭제
- 🔗 **배정 실행 완료 → 항상 자동으로 시트 파라미터 업데이트 + 키워드 시트 기입** (파라미터 유무와 무관, 확인 팝업 없이 순차 실행, 수식 반영 3초 대기)
- 수동 개별 버튼도 유지 (재실행 가능)
- ⚠ 작업 모두 취소 (복원 + ZIP 삭제 + 시트 행 삭제 + written_rows 초기화)

### 🔙 시트 기입 취소 (하단 네비게이션 바 — 전 Step 공통)
- Step 1 시트 기입 완료 후 활성화 → 어느 Step에서든 취소 가능
- 기입한 행 번호를 `written_rows`에 기록 → 역순 삭제
- Step 4 "작업 모두 취소" 시에도 함께 초기화

# ⚠️ 주의사항

- 자사 발행리스트 **B~F열은 수식** → A, N, H열만 프로그램이 기입
- 원고리스트 **G열은 수식** → 건드리지 않음
- 키워드 시트 기입은 **파라미터 기입 후** 실행 (수식 반영 대기)
- EXE 배포 시 `credentials.json` 반드시 같이 배포 (`manual_pass.json`, `exclude_keywords.json`도 함께)
- 이미지 번호는 **30 이하**만 인식 — 원고 본문의 연도(2024), 수치(100) 등은 무시
- G: 공유 드라이브 연결 필요
