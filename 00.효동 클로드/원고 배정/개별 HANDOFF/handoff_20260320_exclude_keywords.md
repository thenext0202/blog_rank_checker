# 원고배정 프로그램 — 업체별 제외 키워드 기능 추가

## 파일 위치 규칙
- 원고배정 관련 handoff 파일은 항상 여기에 저장:
  `C:\Users\iamhy\Desktop\프로그램 개발\00.효동 클로드\원고 배정\`

## 요청 배경
- 특정 업체에 배정하면 안 되는 키워드가 있음
- 예: 긍정애드에 "탈모", "병원" 포함 원고 배정 금지
- 한 번 입력하면 프로그램 재시작 후에도 유지되어야 함

## 변경 파일
- `C:\Users\iamhy\Desktop\프로그램 개발\원고배정\main.py`

## 변경 내용

### 1. 영구 저장 (`exclude_keywords.json`)
- 프로그램 실행 파일 옆에 `exclude_keywords.json` 자동 생성
- 구조: `{"업체명": ["키워드1", "키워드2"], ...}`
- `load_exclusions()` / `save_exclusions()` 함수 추가

### 2. 제외 판정 함수 (`check_exclusion`)
- 업체의 제외 키워드가 원고 **제목** 또는 **폴더명**에 포함되면 해당 키워드 반환
- 대소문자 무시 비교

### 3. 자동 배정 로직 수정 (`_auto_assign`)
- 업체별 점수 계산 시 제외 키워드 매칭되면 해당 업체를 후보에서 `continue`로 스킵
- 제외된 업체 목록을 트리뷰 "제외" 컬럼에 `업체명(키워드)` 형태로 표시

### 4. UI 변경
- **상단 바**에 "제외 키워드 설정" 버튼 배치 (어느 단계에서든 접근 가능)
  - 처음에 Step 3에만 넣었더니 못 찾아서 상단으로 이동
- 다이얼로그: 업체 선택 → 키워드 목록 표시 → 추가(쉼표 구분)/삭제
- Step 3 트리뷰에 "제외" 컬럼 추가 + 빨간 배경(`#FFCDD2`)
- 수동 업체 변경 시에도 제외 키워드 경고 표시

### 5. 수정/삭제
- 인라인 편집 없음 → 삭제 후 재추가 방식
- 사용자 확인 후 현재 방식으로 충분하다고 결정
