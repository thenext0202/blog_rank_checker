## 3. 날짜 추출 (extract_date)

```python
regex: r"(\d{4}-\d{2}-\d{2})_(\d{4}-\d{2}-\d{2})"
```

| 파일명 패턴 | 결과 |
|------------|------|
| `_2026-03-19_2026-03-19.xlsx` | `"2026-03-19"` |
| `_2026-03-16_2026-03-17.xlsx` | `"2026-03-16~2026-03-17"` |

→ A열에 기록될 날짜 문자열

## 5. 데이터 변환 (df_to_rows)

```python
def df_to_rows(date_str, df):
    rows = []
    for _, row in df.iterrows():
        r = [date_str]  # A열에 날짜 삽입
        for v in row:
            if pd.isna(v): r.append("")          # NaN → 빈 문자열
            elif isinstance(v, float) and v == int(v):
                r.append(int(v))                  # 100.0 → 100
            else: r.append(v)
        rows.append(r)
    return rows
```

→ A열(날짜) + B~R열(Excel 데이터) = 18열 리스트

## 6. 평가

**잘된 점:**
- 오늘 수정 파일만 자동 필터 → 실수 방지
- 날짜 범위 표현 지원 (단일/기간)
- NaN, 소수점 정리로 시트에 깔끔한 데이터
- config.json으로 폴더 경로 영속 저장

**부족한 점:**
- Excel 읽기 실패 시 예외 처리 없음
- 파일명에 날짜가 없으면 sys.exit (복구 불가)
- 같은 날짜 파일이 여러 개면 모두 순차 업로드 (마지막 것이 최종)
