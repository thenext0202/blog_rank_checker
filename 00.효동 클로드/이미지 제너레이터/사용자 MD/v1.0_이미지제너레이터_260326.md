# 🖼️ 이미지 제너레이터 v1.0 — 사용 가이드

> **날짜:** 2026-03-26
> **버전:** v1.0
> **경로:** `프로그램 개발\image_generator\main.py`

---

## 📖 이 프로그램은?

원고를 먼저 작성한 뒤, 원고 내용에 어울리는 이미지를 자동으로 찾아주는 프로그램입니다.

> 💡 **왜 분리했나?**
> 기존에는 이미지를 먼저 고르고 원고를 썼는데, 이러면 원고가 이미지에 맞춰져서 부자연스러웠습니다.
> 이제 원고를 자유롭게 쓰고 → 이미지를 원고에 맞춰 선택합니다.

---

## 🚀 사용 방법

### 1단계: 설정

| 항목 | 설명 |
|------|------|
| **Claude API Key** | 장면 추천에 사용 |
| **OpenAI API Key** | 이미지 변형에 사용 (프롬프트 입력 시만) |
| **이미지 시트 ID** | 이미지 메타데이터가 있는 구글 시트 |

> "키 저장" 버튼을 누르면 다음에 다시 입력할 필요 없습니다.

### 2단계: 원고 입력

- **직접 입력**: 왼쪽 텍스트 영역에 원고를 붙여넣기
- **파일 불러오기**: "파일 불러오기" 버튼 → `.txt` 또는 `.docx` 선택

> ⚠️ 원고 안에 이미지 위치가 숫자로 표시되어야 합니다 (줄에 `1`, `2`, `3` 등)

### 3단계: 이미지 자동 제안

1. **제품** 드롭다운에서 제품 선택
2. **"이미지 자동 제안"** 버튼 클릭
3. 오른쪽에 이미지 카드가 표시됩니다

### 4단계: 이미지 확인

각 카드에서:

| 버튼 | 동작 |
|------|------|
| **수락** | 이 이미지로 확정 |
| **다른 이미지 보기** | 대체 이미지 5장 표시 |
| **다시 선택** | 확정을 취소하고 다시 고르기 |

### 5단계: 대체 이미지 / 변형

"다른 이미지 보기"를 누르면 팝업이 뜹니다:

- **대체 이미지 5장** 중에서 "선택" 클릭 → 바로 교체
- **프롬프트 입력** → "변형 실행" 클릭 → AI가 이미지를 새로 만들어줍니다
  - 예: "30대 여성이 거울을 보며 미소짓는 모습으로 변경"

### 6단계: 최종 저장

1. **"최종 저장"** 버튼 클릭
2. 저장할 폴더 선택
3. 결과물:

```
선택한 폴더/
├── 원고.txt     ← 원고 텍스트
├── 1.png        ← 이미지 1번
├── 2.jpg        ← 이미지 2번
├── 3.png        ← 이미지 3번
└── ...
```

> 이미지 파일명은 원고 안의 이미지 번호와 동일합니다.

## ❓ 자주 묻는 질문

### Q: 0번 이미지는 왜 안 나오나요?
> 0번은 대표 이미지로 이미 고정되어 있어서, 1번부터 처리합니다.

### Q: 이미지가 5장도 안 나와요
> 해당 제품의 이미지가 부족하면 "공통" 이미지에서 보충합니다. 시트에 이미지를 더 추가하면 됩니다.

### Q: 미확정 상태로 저장하면?
> 확인 메시지가 뜨고, "예"를 누르면 제안된 이미지로 자동 저장됩니다.

### Q: 프롬프트 변형에 OpenAI Key가 필요한가요?
> 네. 프롬프트로 이미지를 변형/생성할 때만 OpenAI API를 사용합니다. 단순 선택은 필요 없습니다.
