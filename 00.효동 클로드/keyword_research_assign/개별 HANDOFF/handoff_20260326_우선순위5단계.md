# 전광판 배정 우선순위 3→5단계 변경

> 날짜: 2026-03-26
> 파일: `keyword_research_assign\assign.py`

---

## 변경 내용

### 기존 (3단계)
| 우선순위 | 조건 |
|:---:|:---|
| 1차 | 전환금액 있음 + 순위밖 |
| 2차 | 전환금액 있음 + 경과 10일+ |
| 3차 | 순위밖 |

### 변경 후 (5단계)
| 우선순위 | 조건 | 정렬 |
|:---:|:---|:---:|
| 1차 | 기존전환금액(C) >= 50만 AND 순위밖 | C 내림차순 |
| 2차 | 기존전환금액(C) >= 50만 AND 경과일(I) >= 10 AND 전환금액(J) = 0 | C 내림차순 |
| 3차 | 전환금액(J) > 0 AND 순위밖 | C 내림차순 |
| 4차 | 기존전환금액(C) > 0 AND 경과일(I) >= 10 | C 내림차순 |
| 5차 | 순위밖 | C 내림차순 |

---

## 수정 위치 (assign.py)

| 함수 | 변경 |
|:---|:---|
| `load_board_keywords()` | J열(전환금액, col 9) 읽기 추가 → `conversion` 필드 |
| `prioritize_keywords()` | 3단계 → 5단계 분류, 50만원 임계값 적용 |
| `assign_to_writers()` | `priority_labels` 5개로 업데이트 |
| `find_next_keyword()` | `priority_labels` 5개로 업데이트 |
| `main()` | 5개 우선순위 출력 + 튜플 전달 |
