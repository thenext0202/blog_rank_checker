## 1. 기능 요약

시드 키워드를 입력받아 Google/Naver/Daum에서 연관검색어를 수집하고, Naver 검색광고 API로 검색량을 조회하여 시트에 기록한다.

---

## 2. 실행 방법

```bash
python main.py                        # 시트에서 시드 읽기
python main.py 고혈압 혈당관리         # 직접 키워드 입력
python main.py --file seeds.txt       # 파일에서 읽기
```

---

## 3. 처리 흐름

```
① 시트 "시트1" A열(제품명) + B열(시드 키워드) 읽기 (중복 제거)
    ↓
② 각 시드별 연관검색어 수집
    ├── Google Autocomplete (suggestqueries.google.com, JSON)
    ├── Naver 자동완성 (ac.search.naver.com, JSON)
    ├── Naver 연관검색어 (search.naver.com, HTML regex)
    └── Daum 연관검색어 (search.daum.net, HTML regex)
    ↓
③ Naver 검색광고 API로 검색량 조회 (5개씩 배치, 0.3초 간격)
    ↓
④ MIN_VOLUME(500) 이상만 필터링
    ↓
⑤ 시트에 기록: 제품명 | 시드키워드 | 추천키워드 | 출처 | 검색량
```

---

## 4. 시트 구조

| 열 | 내용 | 비고 |
|----|------|------|
| A | 제품명 | 입력 |
| B | 시드 키워드 | 입력 |
| C | 추천 키워드 | 출력 |
| D | 출처 (Google/Naver/Daum) | 출력 |
| E | 검색량 (PC+모바일 합계) | 출력 |

---

## 5. 핵심 함수

| 함수 | 역할 |
|------|------|
| `connect_sheet()` | Google Sheets 연결 |
| `read_seeds_from_sheet()` | A/B열에서 시드 키워드 읽기 |
| `get_google_suggestions(kw)` | Google Autocomplete 호출 |
| `get_naver_suggestions(kw)` | Naver 자동완성 + 연관검색어 |
| `get_daum_suggestions(kw)` | Daum 연관검색어 |
| `call_keyword_api(keywords)` | Naver 검색광고 API (검색량) |
| `run_research(seeds)` | 전체 수집 오케스트레이션 |
| `write_results_to_sheet()` | 결과 시트 기록 |

---

## 6. 설정값

```python
MIN_VOLUME = 500    # 최소 검색량 필터
배치 크기 = 5       # API 호출 단위
```
