## 이 프로그램은 뭘 하나요?

건강/의학 블로그 마케팅에 쓸 **키워드를 자동으로 찾고, 작가에게 배정**해 주는 프로그램입니다.
3개의 독립 프로그램으로 구성되어 있습니다.

| 프로그램 | 하는 일 | 실행 |
|:---:|:---|:---:|
| 🔎 **키워드 리서치** | 시드 키워드로 연관검색어 수집 + 검색량 조회 | `python main.py` |
| 📦 **창고 기반 배정** | 전환/잠재 키워드를 7:3 비율로 작가 배정 | `python keyword_assign.py` |
| 📊 **전광판 기반 배정** | 순위/경과일 기준 우선순위로 작가 배정 | `python assign.py` |

## 🔎 키워드 리서치 (main.py)

시드 키워드를 넣으면 **Google, 네이버, 다음** 3개 플랫폼에서 연관검색어를 수집하고,
네이버 검색광고 API로 **검색량**을 조회해서 시트에 기록합니다.

```
시드: "고혈압"
  → 고혈압 낮추는 방법 (Google) - 검색량 8,500
  → 고혈압 약 부작용 (Naver) - 검색량 3,200
  → 고혈압 좋은 음식 (Daum) - 검색량 12,000
  → 검색량 500 미만은 자동 필터
```

### 실행 방법

```bash
python main.py                    # 시트에서 시드 읽기
python main.py 고혈압 혈당관리     # 직접 입력
```

## 📦 창고 기반 배정 (keyword_assign.py)

키워드를 **전환 키워드**(돈 벌어본 키워드)와 **잠재 키워드**(새로 시도할 키워드)로 나눠서,
**전환 70% + 잠재 30%** 비율로 작가에게 배정합니다.

| 단계 | 설명 |
|:---:|:---|
| 1 | 키워드 창고 갱신 (전환 키워드 + 검색량 1,000+ 잠재 키워드) |
| 2 | 오늘 작가별 배정 건수 확인 |
| 3 | 사전 확정 키워드 입력 (선택) |
| 4 | 배정 실행: 전환 70% → 잠재 30% |
| 5 | 미리보기 → 수동 수정 가능 (대체 후보 5개 제시) |
| 6 | 네이버 스마트블록 체크 |
| 7 | 시트에 기록 |

### 전환점수 산출 (잠재 키워드 평가)

| 항목 | 배점 | 예시 |
|:---|:---:|:---|
| 검색량 | 30점 | 10,000+ = 30점, 5,000+ = 25점 … |
| 구매의도 키워드 | 50점 | "추천", "가격", "효과" 등 포함 |
| 카테고리 매칭 | 15점 | 혈당컷 + "혈당" 키워드 |
| 전환금액 | 20점 | 100,000원+ |

## 📊 전광판 기반 배정 (assign.py)

키워드 전광판의 **순위/경과일 데이터**를 활용해서 우선순위를 매기고 배정합니다.

### 5단계 우선순위 (2026-03-26 업데이트)

> C열 = 기존 전환금액 (누적), J열 = 전환금액 (현재 링크), I열 = 경과일

| 우선순위 | 조건 | 의미 |
|:---:|:---|:---|
| 🔴 **1차** | 기존전환(C) ≥ 50만 + 순위 밖 | 고수익 키워드인데 밀려남 → 복구 최우선 |
| 🟠 **2차** | 기존전환(C) ≥ 50만 + 경과 10일+ + 전환(J) = 0 | 고수익인데 현재 전환 없음 → 갱신 시급 |
| 🟡 **3차** | 전환(J) > 0 + 순위 밖 | 현재 전환 발생 중인데 밀려남 → 복구 |
| 🟢 **4차** | 기존전환(C) > 0 + 경과 10일+ | 전환 이력 있고 오래됨 → 갱신 |
| 🔵 **5차** | 순위 밖 | 나머지 순위 밖 → 신규 진입 시도 |

모든 우선순위 내에서 **기존 전환금액(C열) 내림차순** 정렬

### 작가 매칭 방식

```
점수 = 해당 제품 전환금액 (전문가 우선)
     - 같은 제품 배정 수 × 페널티 (쏠림 방지)
     + 남은 건수 (균등 배분)
→ 점수가 가장 높은 작가에게 배정
```

### 네이버 검색 사전 체크

배정 전에 각 키워드를 네이버에서 검색해서 확인합니다:

| 체크 항목 | 의미 |
|:---|:---|
| 스마트블록 | 블로그 노출에 영향을 줄 수 있는 특수 영역 |
| 인기글 | 인기글 탭 존재 여부 |
| 브랜드 | 브랜드 콘텐츠 영역 존재 여부 |
