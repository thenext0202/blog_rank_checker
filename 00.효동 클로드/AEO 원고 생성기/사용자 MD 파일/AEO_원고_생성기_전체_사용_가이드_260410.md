# AEO 원고 생성기 — 사용 가이드

> 메디셜 AEO 원고를 자동으로 만들어 주는 프로그램입니다.<br>
> **두 가지 방법**으로 쓸 수 있어요. 편한 쪽으로 골라 쓰세요.

<br>

## 🌐 방법 1 — 웹 페이지에서 한 건씩

### 어떻게 쓰는가

1. 브라우저에서 웹 주소로 접속
2. 좌측 폼에서 **제품명 / 카테고리 / 구조 / 채널 / 키워드** 고르거나 입력
3. `원고 생성` 버튼 클릭
4. 30~60초 후 우측에 원고가 나타남
5. 원하면 우측 상단의 `워드 저장` 버튼으로 다운로드

<br>

### 언제 쓰면 좋은가

- 한 건만 빠르게 만들고 싶을 때
- 결과를 그 자리에서 바로 보고 싶을 때

<br>

## 📋 방법 2 — 시트에 직접 입력 (대량 작업용)

### 어떻게 쓰는가

`AEO 발행리스트` 탭에서 빈 행에 다음 **6칸**만 채우면 됩니다.

| 열 | 입력 |
| --- | --- |
| A | 작성일 |
| B | 채널 |
| C | 제품명 |
| D | 카테고리 |
| E | 구조 |
| F | 키워드 |

채우고 그냥 두면 워커가 알아서 잡아서 본문(J열)에 결과를 채워 넣습니다.

<br>

### 시트에서 보이는 흐름

| J열에 보이는 것 | 의미 |
| --- | --- |
| (빈 칸) | 워커 차례 기다리는 중 (최대 60초) |
| `⏳ 생성 중... (14:23:15 시작)` | **워커가 지금 만들고 있음** |
| (실제 본문 텍스트) | ✅ **완료** |
| `❌ 자동 처리 실패 (3회 시도): ...` | ❌ 실패 (J열 직접 비우면 재시도) |

<br>

### 워드 다운로드

처리가 끝나면 **Q열에 `워드 다운로드` 링크가 자동으로 생깁니다.** 클릭하면 그 행의 원고가 워드 파일로 다운로드돼요.

<br>

### 언제 쓰면 좋은가

- 10건, 50건처럼 한꺼번에 많이 만들 때
- 자리 비우고 나중에 결과만 받고 싶을 때

<br>

## ⚠️ 주의할 점

### 시트에 입력할 때

- **A~F 6칸 모두 채워야** 워커가 작동합니다. 한 칸이라도 비어 있으면 안 잡아요
- **G(담당자), H(발행 링크)는 효진님이 직접 채우는 칸**입니다. 워커는 안 건드려요
- **O열(모델)** — 빈 칸이면 Sonnet(기본), 체크하면 Opus(고품질, 느림)
- **드롭다운으로 골라야 하는 칸** (B 채널, C 제품명, D 카테고리, E 구조) — 직접 타이핑하지 말고 셀 클릭해서 드롭다운에서 고르세요

<br>

### 자주 헷갈리는 것

- **C는 제품명, D는 카테고리** — 헷갈리지 말기. 제품명에 카테고리 이름을 넣으면 "카테고리 'XXX' 없음" 에러
- **워커는 60초 주기로 폴링** — 입력하자마자 시작 안 되고 평균 30초 기다림
- **공식 X / 공식 스레드 채널은 자동 분할** — X는 150자, 스레드는 500자씩. 결과는 "포스트 분할" 탭에서 확인

<br>

### ⏳가 5분 이상 그대로 있을 때

워커가 도중에 죽었을 가능성이 있습니다. **그 셀(J열)을 직접 비우면** 다음 폴링에 다시 잡습니다.

<br>

### ❌가 떴을 때

- 마커 안에 짧게 사유가 적혀 있어요. 확인해 보세요
- 다시 시도하고 싶으면 **J열의 ❌ 마커를 직접 비우기**
- 같은 행이 3번 연속 실패하면 ❌가 박히고 더 이상 시도 안 합니다 (직접 비울 때까지)

<br>

### 같은 행 다시 만들고 싶을 때

J열(본문)을 비우면 됩니다. 그러면 워커가 새로 픽업해서 다시 만듭니다.

<br>

### 시트에서 절대 하지 말 것

- **B열 채널 드롭다운 직접 수정 금지** — 효진님이 사전에 색상까지 입혀둔 거라, 코드도 안 건드리는 영역
- **W~Y열 매핑 테이블 건드리지 말기** — 나중에 사용할 데이터
- **T~V열 자사몰/네이버 URL은 효진님이 수정 가능** — 수정하면 다음 원고부터 바로 반영됨

<br>

## 💡 한 줄 요약

| 상황 | 이렇게 |
| --- | --- |
| 한 건만 빠르게 | 웹에서 폼 채우고 생성 |
| 여러 건 한꺼번에 | 시트 A~F 채우고 자리 떠나기 |
| 진행 상황 보고 싶을 때 | 시트 J열 보기 (⏳ = 작업 중) |
| 워드로 받고 싶을 때 | 웹은 "워드 저장" 버튼 / 시트는 Q열 클릭 |
| 다시 만들고 싶을 때 | 그 행의 J열 비우기 |
