## 🚀 실행 방법

### GUI 버전 (추천)
> `run_gui.bat` 더블클릭

- 사무실 화면에 직원들이 앉아있음
- 아래 채팅창에 메시지 입력 → Enter 전송
- 직원 클릭 → 능력 확인 + 직접 대화 가능

### 터미널 버전
> `run.bat` 더블클릭

- 텍스트 기반 채팅
- 명령어: `/팀`, `/민준`, `/수진`, `/현우`, `/지은`, `/태현`, `/메모`, `/리셋`, `/종료`

### 최초 실행 시
- API 키 입력 화면이 나옴
- Anthropic API 키(`sk-ant-...`)를 붙여넣고 Enter
- 한 번 입력하면 `.api_key` 파일에 저장 → 다시 안 물어봄

## 💬 사용 예시

### 기본 모드 (민준이 알아서 배정)
```
치답이: 루테인 블로그 글 써줘
→ 민준: 지은이한테 규정 확인 먼저 시키고, 수진이한테 글 쓰라고 할게요!
→ 지은: (규정 조사 결과)
→ 수진: (블로그 글 완성)
```

### 직접 대화 모드 (GUI)
1. 직원 클릭 → "대화하기" 버튼
2. 해당 직원과 1:1 대화 (민준 거치지 않음)
3. ESC로 기본 모드 복귀

### 직접 대화 모드 (터미널)
```
/수진        → 수진이와 직접 대화 모드 진입
수진에게 메시지 입력...
/팀          → 기본 모드 복귀
```

## 🎮 GUI 화면 설명

```
┌─────────────────────────────────────────────┐
│  [민준방] [수진방] [현우방] [지은방] [태현방]   │
│                                             │
│  [   회의실   ] [  프로젝트실  ] [  CEO실  ] │
├─────────────────────────────────────────────┤
│  채팅 로그                                   │
│  [입력창                         ] [전송]    │
└─────────────────────────────────────────────┘
```

### 직원 상태 표시등 (캐릭터 우상단 점)
- 🟢 초록: 대기 중 (자기 방에 있음)
- 🟡 노랑: 이동 중
- 🟠 주황: 작업 중 (후광 깜빡임)
- 🔵 파랑: 검색 중

### 말풍선
- 직원이 뭘 하고 있는지 캐릭터 위에 표시
- 검색 중이면 검색어가 보임

### 중단 버튼
- 처리 중일 때 "전송" → "중단"(빨간색)으로 바뀜
- 클릭하면 다음 직원 호출을 건너뛰고 중단

## 📝 업무 기록

### 업무 노트 (`/메모`)
- 직원이 작업 완료할 때마다 자동 기록
- 최근 10건 조회 가능
- 날짜, 담당자, 업무 내용, 결과 요약

### 대화 이력
- 프로그램 종료해도 이전 대화 맥락 유지
- `/리셋`으로 초기화 (업무 노트는 유지)

---

## 💰 비용 관련

- 민준(라우팅)과 지은(리서치)은 **Haiku** 사용 → 저렴 (~2원/요청)
- 수진(글쓰기), 현우(광고), 태현(자동화)은 **Sonnet** 사용 (~40원/요청)
- 웹 검색은 DuckDuckGo → **완전 무료**
- 직접 대화 모드로 민준 거치지 않으면 Haiku 비용 절약
