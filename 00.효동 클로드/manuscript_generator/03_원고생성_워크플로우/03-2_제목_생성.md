# 03-2. 제목 생성 (Step 2)

> main.py: `_generate_with_persona()`, `TITLE_PROMPT_TEMPLATE`, `_show_title_dialog()`

---

## 1. 기능 요약

선택된 페르소나 맥락에서 **3개 SEO 제목 후보**를 생성하고 사용자가 선택/편집한다.

---

## 2. 처리 흐름

```
① _generate_with_persona() — 제목 프롬프트 구성
   포함 정보: 페르소나, 제품, 메인키워드, 유형
    ↓
② call_claude_api()
   ├── max_tokens: 1024
   └── 스레드 실행
    ↓
③ _show_title_dialog() — 선택 다이얼로그
   ├── 3개 라디오버튼 카드 ([제목 A/B/C])
   ├── 텍스트 편집 영역
   └── 버튼: "선택 완료" / "제목 없이 생성" / "취소"
    ↓
④ → Step 3 (원고 생성)으로 전달
```

---

## 3. 제목 제약 조건

- **최대 24자** (SEO 최적화)
- 메인 키워드 포함 필수
- 페르소나 맥락 반영

---

## 4. 제목 출력 형식

```
[제목 A]
갱년기 불면증, 멜라토닌이 답일까

[제목 B]
잠 못 드는 50대, 수면제 대신 이것

[제목 C]
불면증 3년차가 찾은 수면 영양제
```

---

## 5. 평가

**잘된 점:**
- 24자 제한으로 네이버 검색결과 노출 최적화
- 페르소나 연결 → 제목-본문 일관성

**부족한 점:**
- 제목 재생성 기능 없음 (3개 마음에 안 들면 처음부터)
- 경쟁 제목 분석 없음 (네이버 기존 상위 제목 참고 불가)
