## 3. 탭 구조

### Tab 1: 원고 생성 (메인)

**상단 — 스크롤 가능 설정 영역:**

| 섹션 | 위젯 |
|------|------|
| 기본 설정 | 제품 드롭다운, 유형 드롭다운, 스타일 드롭다운, 작가명, 날짜 |
| 서식 옵션 | 톤(존댓말/반말), 폰트크기, 정렬, 인용구, 목차 체크, 3회반복 체크 |
| 키워드 | 메인키워드, 연관키워드, 해시태그, nt_medium, 제품링크(자동) |
| 원고 옵션 | 글자수(1500~5100, 300단위), 이미지수 |
| 색상 규칙 | 개선색상, 경고색상, 강조형광, 강조폰트, 제품색상, 제품형광 |
| 추가 지침 | 자유 텍스트 (2줄) |
| 참고자료 | 리스트박스 (자동 매칭), 폴더열기/새로고침 |

**하단:**
- 버튼 바: 원고생성, 연속생성, 미리보기, Word저장, 텍스트저장, 검수, 시트새로고침
- 스코어보드: 글자수/키워드/이미지 (어두운 배경)
- 결과 텍스트: ScrolledText (맑은 고딕 10pt)

### Tab 2: 설정

| 섹션 | 내용 |
|------|------|
| Claude API 키 | 마스킹 입력 + 저장 |
| Google Sheets | 시트 ID 입력 + 저장&연결 + 상태 |
| 폴더 관리 | 참고자료/출력 폴더 열기 |
| 단축키 안내 | 표시만 |
| 현재 설정 미리보기 | 제품/프롬프트/스타일/지침 요약 |
| 버전 정보 | v8.3+ |

### Tab 3: 효율원고 수집

| 위젯 | 역할 |
|------|------|
| 수집 시작 버튼 | Drive에서 효율원고 다운로드 + Claude 분류 |
| 진행률 바 | determinate |
| 상태 텍스트 | 진행 상황 |
| 결과 로그 | 색상별: 성공(초록), 스킵(주황), 에러(빨간) |

### Tab 4: 생성 이력

| 위젯 | 역할 |
|------|------|
| 이력 리스트 | generation_log.json 역순 표시 |
| 새로고침 버튼 | 이력 다시 로드 |
| 초기화 버튼 | 확인 후 이력 삭제 |

## 4. 키보드 단축키

| 단축키 | 동작 |
|--------|------|
| Ctrl+G | 원고 생성 |
| Ctrl+S | Word 저장 |
| Ctrl+Shift+S | 텍스트 저장 |
| Ctrl+P | 프롬프트 미리보기 |
| Ctrl+R / F5 | 시트 새로고침 |

## 6. 평가

**잘된 점:**
- PanedWindow로 설정/결과 영역 비율 조절 가능
- 스코어보드로 핵심 지표 즉시 확인
- 스크롤 가능한 설정 영역 — 많은 옵션을 깔끔하게 배치
- 단축키 지원으로 파워 유저 생산성

**부족한 점:**
- 설정 항목이 너무 많음 — 자주 안 바꾸는 옵션은 접기(collapse) 필요
- 결과 텍스트의 ㄴ서식 하이라이팅이 단순 (실제 Word 미리보기 없음)
- 창 크기/레이아웃 저장 안 됨 (매번 리셋)
