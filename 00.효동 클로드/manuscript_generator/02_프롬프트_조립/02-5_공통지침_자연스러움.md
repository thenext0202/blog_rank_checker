# 02-5. 공통지침 + 줄 끊기 + ㄴ배치 + 자연스러움 + 유형별 재강조

> build_prompt() 섹션 5, 7-2b, 7-2c, 13, 14, 14+

---

## 1. 공통 지침 (섹션 5)

시트 "공통지침" 탭에서 콘텐츠 규칙만 로드.

```
===== 공통 작성 지침 (글 내용 규칙) =====
※ 아래는 '무슨 내용을 쓸지'에 대한 지침입니다.
  서식(이미지, 인용구, 색상, ㄴ규칙)은 별도 섹션에서 지시합니다.
1. {지침1}
2. {지침2}
...
```

**중요:** 서식 관련 규칙은 여기에 넣지 않음. 콘텐츠 규칙만.

## 2. 줄 끊기 규칙 (섹션 7-2b, 고정)

```
===== 줄 끊기 규칙 (매우 중요) =====
- 한 줄 공백 포함 20자 이내
- 의미 단위로 나눠서 여러 줄에 걸쳐 작성

[잘못된 작성]
나는 고혈압이라는 수치를 듣고 너무나도 놀랐다.

[올바른 작성]
나는 고혈압이라는
수치를 듣고
너무나도 놀랐다.
```

```
===== ㄴ 서식 지시 배치 규칙 (매우 중요) =====
```

## 4. 최종 체크리스트 (섹션 13)

공통지침을 프롬프트 끝부분에서 한 번 더 재강조:

```
===== 최종 체크리스트 (반드시 지킬 것) =====
원고 작성 완료 전에 아래 공통지침을 모두 지켰는지 반드시 확인하세요.
하나라도 빠지면 원고가 반려됩니다:
□ 1. {지침1}
□ 2. {지침2}
...
```

## 5. 자연스러움 규칙 (섹션 14, 고정 5가지)

```
===== 자연스러움 규칙 (모든 유형 공통, 매우 중요) =====
이 글은 '정보 전달'이 아니라 '이야기'입니다.
```

| # | 규칙 | 핵심 |
|---|------|------|
| 1 | 성분 설명은 서사 안에서 | 성분 3개 이상 연속 나열 금지, '왜 찾게 됐는지' 먼저 |
| 2 | 에피소드 충분히 전개 | 맘카페/병원/대화 장면 5~8줄 이상, 1줄로 끝내지 말 것 |
| 3 | 대화 인용은 상황묘사와 함께 | 표정, 분위기, 감정 1~2줄 추가 |
| 4 | 감정 흐름 자연스럽게 | 걱정→검색→발견→고민→결심→시도→변화, 한 단계 건너뛰지 말 것 |

## 6. 유형별 재강조 (섹션 14+)

원고 유형에 따라 마지막에 체크리스트 추가:

| 유형 키워드 | 체크 항목 수 | 핵심 |
|------------|-------------|------|
| 제3자 | 6개 | 관찰자≠대상 구분, 대화 5회+, 유머 반전 |
| 수치 충격 | 4개 | 숫자 단독줄, 의사 경고 인용, 역순 구조 |
| 돌발 증상 | 4개 | 탄식 시작, 비유 표현, 내면 독백 |
| 증상 악화 | 3개 | 결말 암시→과거, 구체적 동작, 영양소 전환 서사 |
| 정보 탐색 | 3개 | 선택기준 번호, 오류→수정 구조, 학습자 포지션 |
| 시나리오 | 3개 | 방치 vs 관리 대비, 시점별 감각, 성분나열 금지 |
| 1인칭 경험담 | 3개 | 도입→악화→전환→회복→마무리, 신체감각 |
| 공감 정보 | 3개 | Empathy→Root Cause→Self-Care→Solution, 비유 설명 |

**매칭 방식:** `prompt_type`에 키워드 포함 여부로 판별 (`key in prompt_type`)
매칭 안 되면 일반 재강조 문구 삽입.

## 7. 평가

**잘된 점:**
- 공통지침을 프롬프트 앞+뒤에 배치(섹션 5, 13) → 지시 준수율 향상
- 자연스러움 규칙으로 '정보 나열' 방지 → 서사 중심 원고
- 유형별 재강조로 유형 고유의 핵심 장치 빠뜨리지 않음
- ㄴ배치 규칙에 올바른/잘못된 예시 → Claude 이해도 향상

**부족한 점:**
- 자연스러움 규칙, 줄 끊기 규칙, ㄴ배치 규칙 모두 **하드코딩** → 시트로 이동하면 유연
- 유형별 재강조도 코드 내 dict → 유형 추가 시 코드 수정 필요
- 유형 키워드 매칭이 `in` 연산 → "시나리오"가 "시나리오형(수치)", "시나리오형(질병)" 모두 매칭
