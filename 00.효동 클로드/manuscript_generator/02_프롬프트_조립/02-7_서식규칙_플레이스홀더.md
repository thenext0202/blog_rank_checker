# 02-7. 서식 규칙 + 플레이스홀더

> build_prompt() 섹션 9, Lines 819~841

---

## 1. 기능 요약

시트 "서식규칙" 탭의 템플릿에서 **12개 플레이스홀더**를 GUI 설정값으로 치환하여 프롬프트에 삽입한다.

---

## 2. 플레이스홀더 목록

| 플레이스홀더 | GUI 소스 | 기본값 | 용도 |
|-------------|----------|--------|------|
| `{font_size}` | 글자크기 드롭다운 | 13 | 본문 폰트 크기 |
| `{align_text}` | 정렬 드롭다운 | 가운데정렬 | 텍스트 정렬 |
| `{quote_num}` | 인용구 스타일 | 1 | 인용구 번호 (1~6) |
| `{toc_instruction}` | 목차 체크박스 | 포함 | 목차 포함/제외 지시문 |
| `{title_repeat}` | 3회 반복 체크 | 포함 | SEO 제목 3줄 반복 지시문 |
| `{product_link}` | 자동 생성 | — | 추적 파라미터 포함 링크 |
| `{color_positive}` | 개선 색상 | 파란색 | 긍정/개선 단어 색상 |
| `{color_negative}` | 경고 색상 | 빨간색 | 부정/경고 단어 색상 |
| `{highlight_emphasis}` | 강조 형광펜 | 노란 형광펜 | 강조 부분 하이라이트 |
| `{emphasis_fontsize}` | 강조 글자크기 | 14 | 강조 부분 폰트 크기 |
| `{color_product}` | 제품명 색상 | 청록색 | 제품명 표시 색상 |
| `{highlight_product}` | 제품명 형광펜 | 노란 형광펜 | 제품명 하이라이트 |

## 3. 치환 로직

```python
fmt_template = sheet_data.get("format_instructions") or ""
try:
    result = fmt_template.format(
        font_size=font_size,
        align_text=alignment,
        quote_num=quote_num,
        toc_instruction=toc_instruction,
        product_link=link_text,
        color_positive=color_positive,
        color_negative=color_negative,
        highlight_emphasis=hl_emphasis,
        color_product=color_product,
        highlight_product=hl_product,
        title_repeat=title_repeat_instruction,
        emphasis_fontsize=emphasis_fontsize,
    )
except KeyError as e:
    # 알 수 없는 플레이스홀더 → 오류 메시지
```

---

## 4. 특수 처리

| 조건 | 처리 |
|------|------|
| `highlight_emphasis == "없음"` | `"글꼴 두껍게"`로 대체 |
| `highlight_product == "없음"` | `"글꼴 두껍게"`로 대체 |
| `product_link` 비어있음 | `"(제품 링크)"`로 대체 |
| `title_repeat == True` | SEO 제목 3줄 반복 + ㄴ서식 지시 |
| `title_repeat == False` | "반복하지 않습니다" |

## 6. 평가

**잘된 점:**
- 서식 규칙을 시트에서 관리 → 코드 수정 없이 서식 패턴 변경
- 12개 플레이스홀더로 세밀한 서식 커스터마이징
- `"없음"` 선택 시 자동 fallback (형광펜 → 굵게)
- 검증 함수로 미사용 플레이스홀더 확인

**부족한 점:**
- 플레이스홀더 추가 시 코드 + 시트 양쪽 수정 필요
- 시트의 서식규칙 템플릿이 복잡할 수 있음 (비개발자 편집 어려움)
- KeyError 시 전체 서식규칙이 누락됨 (부분 적용 불가)
