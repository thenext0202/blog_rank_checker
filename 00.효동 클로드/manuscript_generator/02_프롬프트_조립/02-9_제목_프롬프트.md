## 1. 기능 요약

페르소나 + 제품 + 키워드 맥락에서 **SEO 최적화 제목 3개**를 생성하기 위한 프롬프트를 구성한다.

---

## 2. 템플릿 (TITLE_PROMPT_TEMPLATE)

**입력 변수:** product_name, prompt_type, keywords, sub_keywords, persona

**제목 설계 규칙:**
```
- 메인 키워드를 자연스럽게 포함
- 클릭을 유도하는 호기심 자극형 제목
- 네이버 블로그 SEO에 최적화
- 페르소나의 시점/상황이 반영 (제3자면 관찰자 시점)
- 공백 포함 24자 이내
- 서로 다른 스타일로 3개
```

**출력 형식:**
```
[제목 A]
(제목 텍스트)

[제목 B]
(제목 텍스트)

[제목 C]
(제목 텍스트)
```

## 3. build_title_prompt() 파라미터

```python
def build_title_prompt(product_name, prompt_type, keywords, sub_keywords, persona_text):
    return TITLE_PROMPT_TEMPLATE.format(
        product_name=product_name,
        prompt_type=prompt_type,
        keywords=keywords or "(키워드 미입력)",
        sub_keywords=sub_keywords or "(연관 키워드 없음)",
        persona=persona_text or "(페르소나 없음)",
    )
```

- API 호출: max_tokens=1024

## 5. 평가

**잘된 점:**
- 24자 제한으로 네이버 검색결과 노출 최적화
- 페르소나 맥락 연결 → 제목-본문 톤 일관성
- 3개 서로 다른 스타일 → 선택지 다양

**부족한 점:**
- 경쟁 키워드의 기존 상위 제목 분석 기능 없음
- 제목 재생성(다시 3개) 기능이 없음 — 전체 흐름 처음부터 다시
- 템플릿 하드코딩
