## 1. 메인 키워드 (섹션 6)

```
===== 메인 키워드 =====
{keywords}
위 키워드를 원고 제목과 본문에 자연스럽게 포함해주세요.
```

- GUI에서 쉼표 구분 입력
- 제목 + 본문에 자연스럽게 삽입 지시

## 2. 연관 키워드 (섹션 7)

```
===== 연관 키워드 =====
{sub_keywords}
위 연관 키워드도 본문 중간중간에 자연스럽게 녹여주세요.
```

- 본문에 분산 배치 지시

## 6. 제품 링크 자동 생성

```
{base_link}?nt_source=blog&nt_medium={medium}&nt_detail={date}{keyword}&nt_keyword={code}
```

| 파라미터 | 소스 |
|----------|------|
| base_link | 시트 "제품소구점" C열 |
| nt_medium | GUI 입력 |
| nt_detail | 날짜 + 메인키워드 (공백 제거) |
| nt_keyword | 시트 D열 or 하드코딩 코드맵 |

**코드맵 (fallback):**
```
hc=헬리컷, bc=블러드싸이클, gc=글루코컷, sc=상어연골환
pt=퓨어톤 부스트, po=판토오틴, ml=멜라토닌, af=액티플 활성엽산
```

## 7. 평가

**잘된 점:**
- 해시태그를 요청사항 섹션에만 제한 → 본문 자연스러움 유지
- 추적 파라미터 자동 생성으로 마케팅 데이터 수집

**부족한 점:**
- 키워드 밀도 가이드 없음 (몇 회 이상/이하 등)
- 글자수 범위가 300자 단위 고정 → 세밀한 조절 불가
