# 02-3. 작가 스타일 + 톤

> build_prompt() 섹션 2, 3

---

## 1. 작가 스타일 (섹션 2)

시트 "작가스타일" 탭에서 선택된 스타일의 설명을 삽입한다.

```python
style_desc = sheet_data["styles"].get(style_name, "")
parts.append(f"===== 작가 스타일 =====\n{style_desc}")
```

### 5가지 기본 스타일

| 스타일명 | 특징 |
|----------|------|
| 공감형 | 감정적, 여성 건강 중심, 공감 유도 |
| 생활형 | 일상 대화체, 중년 타겟, 편안한 톤 |
| 분석형 | 성분 비교, 데이터 중심, 논리적 |
| 스토리형 | 서사 중심, 직업 기반 에피소드 |
| 감정형 | 감정 여정, 자존감, 내면 독백 |

**스타일 설명에 포함되는 내용:**
- 문체 톤 (예: 다정한, 담담한, 유머러스한)
- 자주 쓰는 표현/키워드
- 문단 구성 습관
- 타겟 독자층

---

## 2. 톤 (섹션 3)

| 톤 | 프롬프트 지시 |
|----|-------------|
| 존댓말 | "~입니다, ~했어요, ~더라고요" / 정중+친근 |
| 반말 | "~거든, ~잖아, ~했더니" / 친구에게 말하듯 편안 |

---

## 3. 평가

**잘된 점:**
- 스타일을 시트에서 관리 → 작가별 스타일 프로필 자유롭게 추가/수정
- 톤 분기로 존댓말/반말 일관성 확보

**부족한 점:**
- 스타일 + 톤 조합의 적합성 검증 없음 (예: 분석형 + 반말이 어울리는지)
- 실제 작가의 기존 원고를 분석해서 스타일을 자동 추출하는 기능 없음
