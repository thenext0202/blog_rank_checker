## 1. 기능 요약

Google Sheets에서 프롬프트/스타일/지침/제품 데이터를 로드하고, Google Drive와 참고자료/샘플 폴더를 동기화한다.

connect_sheet(sheet_id)

## 3. 데이터 로딩 (load_all_from_sheet)

**반환 구조:**
```python
sheet_data = {
    "prompts":            {"유형명": "프롬프트 내용", ...},
    "styles":             {"스타일명": "설명", ...},
    "guidelines":         ["지침1", "지침2", ...],
    "products":           {"제품명": "소구점", ...},
    "product_links":      {"제품명": "URL", ...},
    "product_codes":      {"제품명": "코드", ...},
    "format_instructions": "서식 템플릿 문자열",
    "papers":             {"제품명": ["논문1 요약", ...]}
}
```

### 탭별 로딩 상세

| 탭 | 컬럼 | 로딩 방식 |
|----|------|-----------|
| 프롬프트 | A:유형명, B:내용 | dict(A→B) |
| 작가스타일 | A:스타일명, B:설명 | dict(A→B) |
| 공통지침 | A:번호, B:규칙 | list(B) — 내용 규칙만, 서식 제외 |
| 제품소구점 | A:제품, B:소구점, C:링크, D:코드 | 3개 dict |
| 서식규칙 | A:key(`format_instructions`), B:템플릿 | 단일 문자열 |
| 참고논문 | A:제품, B~F:논문정보 | dict(제품→[요약 리스트]) |

## 4. 참고자료 로딩 (load_refs_for_product)

```
references/
├── 공통/          # 모든 제품 공통 자료
│   ├── guide.txt
│   └── rules.md
└── {제품명}/      # 제품별 전용 자료
    ├── paper1.txt
    ├── info.md
    └── study.pdf
```

- 지원 형식: `.txt`, `.md`, `.csv`, `.docx`, `.pdf`
- 제품 선택 시 자동 매칭 (공통 + 제품별)
- GUI 참고자료 리스트에 표시

---

## 5. 샘플 로딩 (Few-shot)

```
samples/
├── 작가_날짜키워드_시나리오형(수치)_ml.docx
├── 작가_날짜키워드_후기형(ver3)_bc.docx
└── ...
```

- 파일명 패턴: `작가_파라미터_유형_제품코드.docx`
- 같은 제품 + 같은 유형 우선 매칭
- fallback: 같은 유형 다른 제품
- 최대 4,000자 사용 (토큰 최적화)

## 6. Google Drive 동기화

```python
DRIVE_FOLDERS = {
    "references": "1GyB75wDB_A5-7MwvKoYnCf-LrnmivOIM",
    "samples":    "1p0xijew_WL-W7I8QcGwWoW77Xfbzr2wl",
    "output":     "11WrhiUe7vFed2Ep8_Z0vKA9BrMzTcV_I",
}
```

| 동작 | 방향 | 시점 |
|------|------|------|
| references 다운로드 | Drive → 로컬 | 프로그램 시작 (백그라운드) |
| samples 다운로드 | Drive → 로컬 | 프로그램 시작 (백그라운드) |
| output 업로드 | 로컬 → Drive | 파일 저장 후 (백그라운드) |

- 타임스탬프 비교: 로컬이 최신이면 skip
- 상태바에 동기화 진행 표시

## 7. 평가

**잘된 점:**
- 시트 기반으로 비개발자가 프롬프트/지침 직접 수정 가능
- Drive 동기화로 팀원 간 참고자료/샘플 공유 자동화
- 멀티포맷 참고자료 지원 (txt/md/csv/docx/pdf)

**부족한 점:**
- 시트 탭명이 하드코딩 — 탭명 변경 시 코드 수정 필요
- Drive 동기화 충돌 처리 없음 (타임스탬프만 비교)
- 시트 연결 실패 시 오프라인 모드(설정.xlsx) 전환이 자동이 아님
