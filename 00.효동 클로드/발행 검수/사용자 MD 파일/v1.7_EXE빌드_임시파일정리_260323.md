## 📦 사용 방법

### EXE 배포 (다른 사람에게 전달)

```
📂 배포 폴더 구성
  ├─ 발행검수.exe          (47MB, Python 설치 불필요)
  └─ credentials.json      (Google 서비스 계정 인증)
```

1. `발행검수.exe` + `credentials.json` 두 파일을 **같은 폴더**에 넣기
2. `발행검수.exe` 더블클릭으로 실행
3. Python, pip, 라이브러리 설치 **전혀 필요 없음**

> ⚠️ `credentials.json`이 없으면 구글 시트/드라이브 접근 불가 → 프로그램 에러

## 🔧 구현 방법

### 1. EXE 빌드 (PyInstaller)

```bash
python -m PyInstaller --onefile --windowed --name "발행검수" main.py
```

| 옵션 | 설명 |
|------|------|
| `--onefile` | 모든 라이브러리를 단일 EXE로 패키징 |
| `--windowed` | 콘솔 창 없이 GUI만 표시 |

### 2. credentials.json 경로 자동 감지

```python
if getattr(sys, 'frozen', False):
    BASE_DIR = os.path.dirname(sys.executable)  # EXE 위치
else:
    BASE_DIR = os.path.dirname(os.path.abspath(__file__))  # .py 위치

# EXE 옆 → 기존 경로 순서로 탐색
_cred_local = os.path.join(BASE_DIR, "credentials.json")
_cred_orig = os.path.join(os.path.dirname(BASE_DIR), "manuscript_generator", "credentials.json")
CRED_FILE = _cred_local if os.path.exists(_cred_local) else _cred_orig
```

- **EXE 실행 시**: EXE와 같은 폴더의 `credentials.json` 우선 사용
- **Python 실행 시**: 기존 `manuscript_generator/credentials.json` 경로 그대로 사용
- 둘 다 호환됨

### 3. 임시파일 자동 정리

기존에는 `tempfile.mkdtemp()`로 다운로드한 DOCX 파일이 `AppData\Local\Temp\`에 계속 쌓였음.

```python
# 다운로드 시 임시 폴더 경로 수집
tmp_dirs = []
path, tmp_dir = find_and_download(creds, fn)
if tmp_dir:
    tmp_dirs.append(tmp_dir)

# 검수 완료 후 일괄 삭제
def _cleanup_tmp(self, tmp_dirs):
    for d in tmp_dirs:
        shutil.rmtree(d, ignore_errors=True)
```

| 시점 | 동작 |
|------|------|
| 검수 정상 완료 | `tmp_dirs` 전체 삭제 |
| 중간 중단 (중지 버튼) | 그때까지 다운로드한 임시 폴더 삭제 |
| 파일 못 찾음 | `tmp_dir=None` → 리스트에 추가 안 함 |
