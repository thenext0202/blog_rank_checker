# 📋 댓글 검수 v2.1 업데이트 — 개별 HANDOFF

> 📅 날짜: 2026-03-20
> 📁 수정 파일: `댓글검수\main.py`

---

## 🎯 변경 요약

v1.0(링크별 그룹 방식) → v2.1(독립 목록 + 병렬 스크래핑)으로 전면 리팩터링

---

## 🔧 변경 내용

### 1. 입력 방식 변경
| 항목 | v1.0 | v2.1 |
|:---|:---|:---|
| 링크 입력 | A열에 링크, 아래에 소속 댓글 묶음 | 링크열에 링크만 쭉 나열 |
| 댓글 입력 | 각 링크별로 댓글 따로 입력 | 댓글열에 공통 댓글만 쭉 나열 |
| 체크 방식 | 링크별로 소속 댓글만 체크 | **모든 링크 × 모든 댓글** |
| 결과 출력 | 입력 행 그대로 결과 기록 | **묶음 형태로 자동 재배치** |

### 2. 5블록 지원
- 기존: A-E 1블록만 사용
- 변경: A-E, F-J, K-O, P-T, U-Y **5블록** 동시 지원
- 데이터 있는 블록만 자동 인식

### 3. 병렬 스크래핑
- 기존: `create_driver()` 1개로 순차 처리
- 변경: `ThreadPoolExecutor(max_workers=4)` + URL별 독립 브라우저
- 고유 URL 추출 → 4개 동시 스크래핑 → 결과 캐시 → 블록별 매칭
- **약 4배 속도 향상** (80개 URL 기준 ~10분)

### 4. 시트 자동 확장
- 결과 행 수가 시트 최대 행 수를 초과하면 `ws.resize()` 자동 호출

### 5. 미등록 댓글 내용 기록
- 기존: E열에 표시만
- 변경: E열(각 블록 5번째 열)에 **댓글 내용 그대로** 기록

### 6. 탭 이름 변경
- `시트23` → `댓글 비교`
