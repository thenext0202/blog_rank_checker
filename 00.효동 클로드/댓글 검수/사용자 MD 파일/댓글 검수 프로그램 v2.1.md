## 📌 이 프로그램은 뭘 하나요?

네이버 블로그에 달린 **댓글을 자동으로 검수**해 주는 프로그램입니다.

구글 시트에 **블로그 링크 목록**과 **들어가야 할 댓글 목록**을 따로 넣어 놓으면,
프로그램이 **모든 링크를 모든 댓글로 체크**하고 결과를 묶음 형태로 정리해 줍니다.

| 🏷️ 검수 항목 | 📝 설명 | 📊 시트 표기 |
|:---:|:---|:---:|
| ✅ **정상** | 댓글이 블로그에 잘 달려 있음 | 빈칸 |
| 🔴 **중복** | 같은 댓글이 2번 이상 달림 | 3열: `중복(2회)` |
| ⚠️ **누락** | 시트에 있는 댓글이 블로그에 없음 | 4열: `○` |
| 🟡 **미등록** | 블로그에 있지만 시트에 없는 댓글 | 5열: 댓글 내용 그대로 |

## ⚙️ 기술적으로 어떻게 동작하나요?

### 1단계: 시트 파싱 📊
- 구글 시트에서 전체 데이터를 읽어옴
- 5개 블록(A-E, F-J, K-O, P-T, U-Y)을 순회하며 데이터 있는 블록 감지
- 각 블록에서 링크 목록과 댓글 목록을 독립적으로 추출

### 2단계: 병렬 스크래핑 🚄
- 전체 블록의 고유 URL을 한꺼번에 모음 (중복 제거)
- **브라우저 4개 동시 실행**으로 4배 빠른 수집
- 동일 URL이 여러 블록에 있어도 1번만 스크래핑 (캐시)

### 3단계: 매칭 비교 🔎
- 공백을 모두 제거한 후 비교 (띄어쓰기 차이 무시)
- **10글자 이상 연속으로 겹치면** 같은 댓글로 판정
- 시트 댓글과 블로그 댓글이 미묘하게 달라도 매칭 가능

### 4단계: 결과 기록 ✍️
- 시트 크기가 부족하면 자동 확장
- 기존 데이터 클리어 후 묶음 형태로 재배치
- 미등록 댓글은 실제 댓글 내용을 그대로 기록

## 🔧 설정 변경이 필요할 때

`main.py` 상단의 설정값:

```python
SPREADSHEET_ID = "14IQ3of3Pe9TI-VNHAKNisxYLNub9DSSH7ZZx2Rnzbj8"  # 시트 ID
SHEET_TAB = "댓글 비교"                                              # 탭 이름
MATCH_MIN_LEN = 10                                                  # 매칭 최소 글자 수
NUM_WORKERS = 4                                                     # 병렬 브라우저 수
```

- **다른 시트** 사용: `SPREADSHEET_ID` 변경 + 서비스 계정 공유
- **다른 탭** 사용: `SHEET_TAB` 변경
- **매칭 민감도** 조절: `MATCH_MIN_LEN` 숫자를 줄이면 더 느슨하게, 늘리면 더 엄격하게
- **속도 조절**: `NUM_WORKERS` 숫자를 늘리면 더 빠르지만 PC 부담 증가
