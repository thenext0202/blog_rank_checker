## 📌 이 프로그램은 뭘 하나요?

네이버 블로그에 달린 **댓글을 자동으로 검수**해 주는 프로그램입니다.

구글 시트에 **블로그 링크**와 **들어가야 할 댓글 목록**을 넣어 놓으면,
프로그램이 자동으로 블로그에 접속해서 아래 3가지를 확인합니다:

| 🏷️ 검수 항목 | 📝 설명 | 📊 시트 표기 |
|:---:|:---|:---:|
| ✅ **정상** | 댓글이 블로그에 잘 달려 있음 | 빈칸 |
| 🔴 **중복** | 같은 댓글이 2번 이상 달림 | C열: `중복(2회)` |
| ⚠️ **누락** | 시트에 있는 댓글이 블로그에 없음 | D열: `○` |
| 🟡 **미등록** | 블로그에 있지만 시트에 없는 댓글 | E열: 댓글 텍스트 |

## 📋 구글 시트 사용법

### 시트 구조

```
     A열              B열              C열        D열        E열
┌──────────────┬──────────────────┬──────────┬──────────┬──────────────┐
│ 블로그 링크    │ 댓글              │ 중복 여부  │ 누락 여부  │ 미등록 댓글    │
├──────────────┼──────────────────┼──────────┼──────────┼──────────────┤
│ https://...  │ 좋은 정보 감사합니다  │          │          │ (자동 기입)   │
│              │ 저도 같은 경험이...  │          │ ○        │ (자동 기입)   │
│              │ 위염 때문에 힘들어요  │ 중복(2회)  │          │              │
├──────────────┼──────────────────┼──────────┼──────────┼──────────────┤
│ https://...  │ 다음 블로그 댓글1   │          │          │              │
│              │ 다음 블로그 댓글2   │          │          │              │
└──────────────┴──────────────────┴──────────┴──────────┴──────────────┘
```

### 💡 핵심 규칙

1. **1행은 헤더** — 프로그램이 자동으로 건너뜁니다
2. **A열에 링크가 있으면** → 새 블로그 그룹 시작
3. **A열이 비어있고 B열에 댓글이 있으면** → 바로 위 링크에 소속된 댓글
4. **C, D, E열은 건드리지 마세요** → 프로그램이 자동으로 채워줍니다

### ⚠️ 주의사항

- A열에 링크 말고 다른 값(번호 등)을 넣으면 그걸 링크로 인식해서 오류 발생!
- 시트에 편집 권한이 있어야 합니다 (서비스 계정 공유 필수)

## ⚙️ 기술적으로 어떻게 동작하나요?

### 1단계: 시트 파싱 📊
- 구글 시트에서 전체 데이터를 읽어옴
- A열 기준으로 블로그 링크별 **그룹** 자동 생성
- 각 그룹에 소속된 B열 댓글들을 묶어서 관리

### 2단계: 블로그 댓글 수집 🌐
- Selenium(Chrome 자동화)으로 네이버 블로그 접속
- 네이버 블로그는 **iframe 구조**라서 `mainFrame`으로 전환 필요
- 댓글 영역 스크롤 → **"더보기" 버튼 반복 클릭** → 모든 댓글 로드
- 각 댓글의 텍스트를 추출해서 리스트로 저장

### 3단계: 매칭 비교 🔎
- 공백을 모두 제거한 후 비교 (띄어쓰기 차이 무시)
- **10글자 이상 연속으로 겹치면** 같은 댓글로 판정
- 이렇게 하는 이유: 시트 댓글과 실제 블로그 댓글이 미묘하게 다를 수 있어서
  (예: 끝부분이 잘리거나, 특수문자가 다르거나)

### 4단계: 결과 기록 ✍️
- C열(중복), D열(누락), E열(미등록)에 일괄 업데이트
- 블로그 링크당 1번만 접속해서 효율적
